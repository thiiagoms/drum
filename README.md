### Dependencies

- Web Browser (Firefox, Chrome, etc)

### Run:

01 - Clone:
```bash
$ git clone https://github.com/thiiagoms/drum
```

02 - Vá para o diretório do projeto
```bash
$ cd drum
drum $
```

03 - Open  `index.html` file with your favorite browser:
```bash
drum $ firefox index.html
```
